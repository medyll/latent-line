**Sprint:** 30  
**Points:** 4

## Description

Implement advanced search functionality with full-text search across all fields, multiple filter criteria, and saved search presets.

---

## Features

### Full-Text Search
- Search across timeline events (prompts, notes, actions)
- Search across characters (name, voice_id, outfit prompts)
- Search across environments (prompt, ref)
- Search across config (checkpoint, sampler, tts_engine)
- Debounced input (300ms)

### Filters
- Filter by time range (start-end ms)
- Filter by mood (joyful, melancholic, anxious, serene, curious)
- Filter by lighting type (dusk, daylight, studio, tungsten, ambient)
- Filter by character ID
- Filter by asset usage (has character, has camera, has audio, etc.)

### Saved Presets
- Save current search + filters as preset
- Load preset with one click
- Delete preset
- Export/import presets as JSON

### UI
- Search panel (collapsible)
- Keyboard shortcut (Ctrl+F to open)
- Results highlight matching text
- Jump to event on result click

### Search Index Structure

```typescript
interface SearchIndex {
  events: Map<number, IndexedEvent>;
  characters: Map<string, IndexedCharacter>;
  environments: Map<string, IndexedEnvironment>;
}

interface IndexedEvent {
  time: number;
  text: string; // Concatenated searchable text
  event: TimelineEvent;
}
```

### Debounce Implementation

```typescript
function useDebounce<T>(fn: (value: T) => void, delay: number) {
  let timer: ReturnType<typeof setTimeout> | null = null;
  return (value: T) => {
    if (timer) clearTimeout(timer);
    timer = setTimeout(() => fn(value), delay);
  };
}
```
