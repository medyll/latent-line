# S18-04 — Export ZIP (JSON + metadata)

**Sprint:** 18
**Points:** 3
**Priorité:** 🟡 Moyenne

## Contexte

Pour archiver ou partager un projet complet, l'utilisateur a besoin d'un bundle unique contenant le modèle JSON, les prompts exportés et les métadonnées du projet.

## Objectif

Générer un fichier ZIP contenant tous les artefacts exportables du projet.

## Critères d'acceptance

- [ ] Bouton "Export ZIP" dans le menu Export
- [ ] Contenu du ZIP : `model.json`, `prompts.txt`, `prompts.json`, `events.csv`, `README.txt` (résumé projet)
- [ ] `README.txt` : nom projet, date, nombre d'events, FPS, résolution
- [ ] Génération 100% client-side
- [ ] Nommage : `{projet}-{date}.zip`

## Implémentation suggérée

- Librairie `jszip` (léger, client-side)
- `src/lib/utils/export-zip.ts`
- Réutilise les fonctions des stories S18-01/02/03
