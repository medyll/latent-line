# S23-03 — Historique de versions avec labels

**Sprint:** 23
**Points:** 8
**Priorité:** 🟡 Moyenne

## Contexte

L'undo/redo couvre les actions récentes mais pas les sauvegardes de jalons ("v1 - avant refonte", "v2 - révision client"). Il n'y a pas de moyen de revenir à un état antérieur de la veille.

## Objectif

Permettre de créer des sauvegardes nommées du projet et d'y revenir.

## Critères d'acceptance

- [ ] Bouton "Créer un snapshot" → invite à nommer la version
- [ ] Liste des snapshots dans un panneau "Historique" (nom, date, nb d'events)
- [ ] Clic sur un snapshot → confirmation puis restauration
- [ ] Maximum 20 snapshots stockés (LRU si dépassé)
- [ ] Snapshots persistés dans `localStorage['latent-line:snapshots']`
- [ ] Export d'un snapshot comme fichier JSON indépendant

## Implémentation suggérée

- `src/lib/stores/snapshots.svelte.ts`
- `VersionHistory.svelte` — panneau latéral
- Format : `{ id, label, date, model: Model }`
