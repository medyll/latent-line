## Technical Notes

### JSON-LD Schema
```jsonld
{
  "@context": "https://schema.org/",
  "@type": "CreativeWork",
  "name": "Timeline Name",
  "description": "...",
  "creator": "latent-line",
  "dateCreated": "ISO 8601",
  "hasPart": [
    {
      "@type": "Action",
      "startTime": "PT00:10:00S",
      "duration": "PT00:05:00S",
      "object": { "@type": "Asset", "name": "..." }
    }
  ]
}
```

### YAML Format
```yaml
name: My Timeline
assets:
  characters:
    - id: char_1
      name: Alice
      outfits: { default: "..." }
  environments: [...]
timeline:
  events:
    - id: evt_1
      time: 1000
      label: "Scene 1"
      assets: [char_1, env_1]
```
