# S16-01 — Preview frame au survol d'un event

**Sprint:** 16
**Points:** 3
**Priorité:** 🟠 Haute

## Contexte

Les events dans la timeline affichent peu d'informations visuelles. L'utilisateur doit cliquer pour voir les détails. Un aperçu rapide au survol accélérerait la navigation.

## Objectif

Afficher une tooltip riche (prompt, personnage, mood) au survol d'un event timeline.

## Critères d'acceptance

- [ ] Tooltip apparaît 300ms après le survol d'un event (synoptic ou temporal)
- [ ] Affiche : numéro d'event, frame, personnage, mood, action (tronquée à 80 chars)
- [ ] Se ferme immédiatement au mouseLeave
- [ ] Positionnement intelligent (ne sort pas du viewport)
- [ ] Accessible : `role="tooltip"`, `aria-describedby`

## Implémentation suggérée

- `src/lib/components/app/EventTooltip.svelte` — composant tooltip
- `use:tooltip` action Svelte pour attacher aux event cards
- CSS `position: fixed` avec calcul de position via `getBoundingClientRect`
