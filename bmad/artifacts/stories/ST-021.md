# ST-021 — Temporal Sequencer: drag, zoom, playhead

- sprint: 5
- created: 20260308T
- status: done
- points: 5

## Description

Implement interactivity in the Temporal Sequencer. Events are positioned absolutely at their time offset. Drag horizontally to change time (TS-03). Zoom slider scales pixel density (TS-04). Playhead indicator at scrubable position (TS-05).

## Acceptance Criteria

- Events positioned absolutely: left = time × FRAME_PX × zoom
- Drag (pointer events) changes event.time; no collision allowed (snaps to nearest free slot)
- Zoom slider (1–8×) scales all event positions and widths
- Playhead: vertical line at playheadTime, updated by clicking timeline
- Unit tests for time-from-drag calculation and collision resolution

## Implementation Notes

- `SequenceOrchestrator.svelte` extended with full temporal sequencer section
- `FRAME_PX = 24`, `EVENT_WIDTH = 56`; `pixelsPerFrame = $derived(FRAME_PX * zoomLevel)`
- `containerWidth = $derived(Math.max(600, (maxTime + 20) * pixelsPerFrame + EVENT_WIDTH + 40))`
- Absolute-positioned event blocks: `left: event.time * pixelsPerFrame`; collision-resolved on drag end
- Drag via pointer capture: `startDrag` calls `setPointerCapture`, `onDragMove` computes preview, `onDragEnd` resolves collision and commits to `model.timeline[idx].time`
- Collision resolution: `while (ev.time !== startTime && ev.time === resolved) resolved++`
- Zoom slider: `<input type="range" min=1 max=8 step=0.5 bind:value={zoomLevel}>` in CardHeader
- Playhead: red `w-px` div positioned at `playheadTime * pixelsPerFrame`; set by click on outer timeline div
- Time grid ticks every 10 frames rendered as light gray marks
- `TemporalSequencer.test.ts`: 14 tests — drag calculation (5), collision resolution (4), zoom (3), playhead (2) — all passing
