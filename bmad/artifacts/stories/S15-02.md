# S15-02 — Configuration projet persistante

**Sprint:** 15
**Points:** 3
**Priorité:** 🟡 Moyenne

## Contexte

Le modèle a un champ `config` (FPS, résolution) mais il n'existe pas d'UI dédiée pour l'éditer. Les réglages sont perdus si le champ n'est pas exposé.

## Objectif

Créer un panneau de configuration projet éditable et persisté avec le modèle.

## Critères d'acceptance

- [ ] Panneau "Projet" avec : nom du projet, FPS (12/24/30/60), résolution (720p/1080p/custom), durée totale
- [ ] Les valeurs sont lues/écrites dans `model.config`
- [ ] Les changements déclenchent la sauvegarde localStorage existante
- [ ] Validation : FPS entre 1 et 120, résolution > 0
- [ ] Accessible depuis la topbar

## Implémentation suggérée

- `src/lib/components/app/ProjectConfig.svelte`
- Étendre `model-types.ts` → champs `name: string`, `fps: number`, `width/height: number`
- Zod schema mis à jour
