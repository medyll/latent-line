**Sprint:** 29  
**Points:** 3  
**Status:** Pending  
**Priority:** P0

---

## Description

Allow users to select multiple timeline events and perform operations on all of them simultaneously.

---

## Features

### Multi-Select
- **Ctrl+Click** — Toggle single event selection
- **Shift+Click** — Select range
- **Drag marquee** — Draw selection box
- **Ctrl+A** — Select all

### Bulk Operations
- **Delete** — Remove all selected events
- **Duplicate** — Copy all selected events
- **Move** — Shift time for all selected
- **Batch edit** — Edit common properties

---

## Acceptance Criteria

- [ ] Multi-select with modifiers works
- [ ] Visual feedback for selection
- [ ] Bulk operations apply to all
- [ ] Undo/Redo works for bulk ops
- [ ] Unit tests (8+)
