# S24-02 — Export prompts format Deforum

**Sprint:** 24
**Points:** 5
**Priorité:** 🟠 Haute

## Contexte

Deforum (extension SD WebUI) est l'outil le plus utilisé pour la vidéo AI. Il attend les prompts dans un format JSON spécifique avec clés de frames.

## Objectif

Générer un export directement compatible avec Deforum SD WebUI.

## Critères d'acceptance

- [ ] Export "Deforum JSON" dans le menu Export
- [ ] Format : `{ "prompts": { "0": "prompt...", "24": "prompt..." }, "negative_prompts": {...} }`
- [ ] Les frames de transition interpolés (entre deux events) reçoivent un prompt de morphing
- [ ] Options : négatif custom global, seed, steps, cfg_scale
- [ ] Preview du JSON avant export
- [ ] Validation : frame 0 toujours présent (requis par Deforum)

## Implémentation suggérée

- `src/lib/utils/export-deforum.ts` — fonction pure `toDeforumFormat(model, options)`
- Tests unitaires avec snapshot du JSON généré
- Réutilise `export-prompts.ts` de S18-03
