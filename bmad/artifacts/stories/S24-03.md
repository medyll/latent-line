# S24-03 — Export format FramePack / CogVideoX

**Sprint:** 24
**Points:** 5
**Priorité:** 🟡 Moyenne

## Contexte

FramePack et CogVideoX sont des outils de génération vidéo AI émergents avec leurs propres formats de séquence. Supporter ces formats positionne latent-line comme hub central.

## Objectif

Exporter la séquence dans les formats FramePack et CogVideoX.

## Critères d'acceptance

- [ ] Export "FramePack" : JSONL avec un objet par frame `{ frame, prompt, character_ref, ... }`
- [ ] Export "CogVideoX" : script de frames avec interpolation de keyframes
- [ ] Les deux formats incluent les données de caméra (zoom, pan, tilt)
- [ ] Documentation inline des formats dans `src/lib/utils/export-*.ts`
- [ ] Tests unitaires avec fixtures des formats attendus

## Implémentation suggérée

- `src/lib/utils/export-framepack.ts`
- `src/lib/utils/export-cogvideo.ts`
- Constante `FORMAT_VERSION` dans chaque module pour versionner les formats
