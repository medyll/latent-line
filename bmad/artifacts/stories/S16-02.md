# S16-02 — Lecteur de prévisualisation avec transport

**Sprint:** 16
**Points:** 8
**Priorité:** 🟠 Haute

## Contexte

Le moteur de playback existe (`playback.ts`) mais il n'y a pas de prévisualisation visuelle des events pendant la lecture. L'utilisateur ne peut pas "voir" son animation se dérouler.

## Objectif

Ajouter un panneau de prévisualisation qui affiche l'event courant en temps réel pendant la lecture.

## Critères d'acceptance

- [ ] Zone "Preview" dans le panneau droit (ou modal plein écran)
- [ ] Pendant la lecture, affiche l'event actif : personnage, action, mood, environnement
- [ ] Highlight visuel de l'event actif dans la synoptic view
- [ ] Indicateur de progression dans le temporal sequencer synchronisé
- [ ] Boutons Play/Pause/Stop dans la preview également
- [ ] Mode plein écran (touche F ou bouton)

## Implémentation suggérée

- `src/lib/components/app/PreviewPanel.svelte`
- Abonnement au store de playback existant
- CSS transition entre events (crossfade 100ms)
