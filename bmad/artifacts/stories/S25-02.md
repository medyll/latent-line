# S25-02 — Recherche globale dans le projet

**Sprint:** 25
**Points:** 3
**Priorité:** 🟠 Haute

## Contexte

Sur les projets de 50+ events, retrouver un event spécifique par son contenu (texte d'action, personnage) est fastidieux sans recherche.

## Objectif

Ajouter une recherche globale (Ctrl+F) qui filtre events ET assets simultanément.

## Critères d'acceptance

- [ ] Ctrl+F ouvre un overlay de recherche global
- [ ] Recherche en temps réel dans : action, speech, mood, personnage, environnement, notes
- [ ] Résultats groupés : Events (N) | Personnages (N) | Environnements (N) | Audio (N)
- [ ] Clic sur un résultat → sélectionne et scroll vers l'élément
- [ ] Escape ferme le panneau
- [ ] Highlight du terme recherché dans les résultats

## Implémentation suggérée

- `GlobalSearch.svelte` — overlay modal avec input + résultats
- `src/lib/utils/search.ts` — fonction pure `searchModel(model, query)` retournant des `SearchResult[]`
- Tests unitaires sur `searchModel`
