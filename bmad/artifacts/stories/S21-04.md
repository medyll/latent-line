# S21-04 — Panneau redimensionnable (drag-to-resize)

**Sprint:** 21
**Points:** 5
**Priorité:** 🟡 Moyenne

## Contexte

Les largeurs des panneaux latéraux sont fixes. Certains utilisateurs veulent plus d'espace pour la timeline, d'autres pour les assets.

## Objectif

Permettre de redimensionner les panneaux par drag sur le bord séparateur.

## Critères d'acceptance

- [ ] Séparateur draggable entre AssetManager et workspace
- [ ] Séparateur draggable entre workspace et PropertiesPanel
- [ ] Taille min/max par panneau (min 160px, max 50% viewport)
- [ ] Position mémorisée dans `localStorage['latent-line:layout']`
- [ ] Double-clic sur le séparateur remet la taille par défaut
- [ ] Cursor `col-resize` sur le séparateur

## Implémentation suggérée

- Action Svelte `use:resizable` dans `src/lib/actions/resizable.ts`
- Pointer events (mousedown/pointermove/pointerup) pour compatibilité touch
