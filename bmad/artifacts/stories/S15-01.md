# S15-01 — Préférences utilisateur persistantes

**Sprint:** 15
**Points:** 3
**Priorité:** 🟡 Moyenne

## Contexte

Dark mode, zoom par défaut et autres préférences visuelles sont réinitialisés à chaque rechargement. Seul le thème est partiellement persisté.

## Objectif

Centraliser et persister toutes les préférences utilisateur dans `localStorage` via un store dédié.

## Critères d'acceptance

- [ ] Store `preferencesStore` (`$state`) avec : `theme`, `defaultZoom`, `sidebarWidth`, `language`
- [ ] Chargement au démarrage depuis `localStorage['latent-line:prefs']`
- [ ] Sauvegarde automatique via `$effect` à chaque changement
- [ ] Panneau "Préférences" accessible depuis la topbar (icône ⚙)
- [ ] Réinitialisation possible ("Reset to defaults")
- [ ] Tests unitaires sur le store

## Implémentation suggérée

- `src/lib/stores/preferences.svelte.ts` — store + persistence
- `src/lib/components/app/PreferencesPanel.svelte` — UI panneau
- Intégration dans `+layout.svelte` pour propagation globale
