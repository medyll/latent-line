# S10-02 — Export Timeline JSON

**Sprint**: 10 | **Points**: 3 | **Status**: done ✅ (already implemented in SystemFooter.svelte)

## Description

As a developer, I can export the current timeline model as a JSON file, so I can use it downstream or back it up.

## Acceptance Criteria

- An "Export JSON" button is visible in the UI (SystemFooter or PropertiesPanel header)
- Clicking it triggers a browser file download named `timeline-<timestamp>.json`
- The downloaded JSON is the full `Model` object, pretty-printed (2-space indent)
- The export is purely client-side (no server call)

## Technical Notes

```ts
function exportModel(model: Model) {
  const blob = new Blob([JSON.stringify(model, null, 2)], { type: 'application/json' });
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = `timeline-${Date.now()}.json`;
  a.click();
  URL.revokeObjectURL(url);
}
```
