# S23-02 — Commentaires sur les events

**Sprint:** 23
**Points:** 5
**Priorité:** 🟡 Moyenne

## Contexte

En collaboration, les équipes ont besoin d'annoter les events avec des retours de production ("changer le mood", "vérifier le lip sync") sans modifier les données de prompt.

## Objectif

Ajouter un champ commentaire par event, visible dans la PropertiesPanel.

## Critères d'acceptance

- [ ] Champ `notes: string` optionnel dans le modèle d'event (`frame.notes`)
- [ ] Zone de texte multiligne dans PropertiesPanel (section "Notes de production")
- [ ] Icône "bulle" sur les events cards qui ont des notes (indicateur visuel)
- [ ] Export CSV et JSON incluent les notes
- [ ] Les notes sont incluses dans l'undo/redo
- [ ] Recherche globale filtre aussi les notes

## Implémentation suggérée

- Étendre `Frame` dans `model-types.ts` avec `notes?: string`
- Zod schema mis à jour
- `NotesField.svelte` dans PropertiesPanel
