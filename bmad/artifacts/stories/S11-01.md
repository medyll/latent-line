# S11-01 — Intégrer ModelInspector

**Sprint**: 11 | **Points**: 3 | **Status**: done ✅

## Description

As a developer, I can open a collapsible ModelInspector panel to see the current model as JSON with inline Zod validation status, so I can debug models without exporting.

## Acceptance Criteria

- A toggle button in the app (e.g. footer or top-bar) opens/closes the ModelInspector overlay or panel
- ModelInspector displays the full `Model` as formatted JSON
- Zod validation result (pass/fail + error list) is shown above the JSON
- Panel is collapsible/dismissible (Escape key or X button)
- Component reads `MODEL_STORE_KEY` from context (no prop drilling)

## Technical Notes

- `ModelInspector.svelte` already exists — read it first, extend rather than rewrite
- A `$state showInspector = false` on the page, passed as a prop or via context
- Use `<pre>` + `JSON.stringify(model, null, 2)` for the JSON view
- Position: fixed overlay (bottom-right) or a fourth resizable panel
