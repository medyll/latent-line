# S10-01 — Playback Engine

**Sprint**: 10 | **Points**: 5 | **Status**: done ✅

## Description

As a content creator, I can play/pause/stop the timeline and see a playhead move in real-time across the TemporalSequencer, so I can preview event timing visually.

## Acceptance Criteria

- A `PlaybackBar` component (or SystemFooter extension) exposes Play / Pause / Stop buttons
- A reactive `playheadTime` state (ms) is shared via context or prop-drilled to TemporalSequencer
- While playing, `playheadTime` increments at real-time speed (requestAnimationFrame loop)
- Stop resets `playheadTime` to 0
- Playhead is a vertical line rendered over the timeline canvas at the correct pixel offset (`playheadTime / MS_PER_PX`)
- Clicking anywhere on the TemporalSequencer track area scrubs to that time position (mousedown x → time)
- Keyboard: Space = play/pause, Escape/0 = stop

## Technical Notes

- Use a Svelte `$state` for `playheadTime` + `isPlaying` in the page store or a new `playback.svelte.ts` module
- rAF loop: `let raf; function tick(ts) { playheadTime = ...; raf = requestAnimationFrame(tick); }`
- Cancel rAF on component destroy or stop
- `FRAME_PX = 24`, `MS_PER_FRAME = 1000/24` → `MS_PER_PX = MS_PER_FRAME / FRAME_PX`
- Playhead must stay visible (overflow clip on sequencer track, horizontal scroll follows playhead)
