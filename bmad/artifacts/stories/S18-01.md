# S18-01 — Export storyboard PDF

**Sprint:** 18
**Points:** 8
**Priorité:** 🟠 Haute

## Contexte

Les équipes de production ont besoin d'un document imprimable résumant la séquence. Un storyboard PDF est le livrable standard de pré-production.

## Objectif

Générer un PDF du storyboard avec une card par event : numéro, personnage, action, mood, durée.

## Critères d'acceptance

- [ ] Bouton "Export PDF" dans la topbar
- [ ] PDF généré côté client (pas de serveur)
- [ ] Layout : grille 2 colonnes, une card par event
- [ ] Chaque card : numéro, frame start-end, personnage, mood (couleur), action (tronquée 120 chars), notes
- [ ] Page de garde : titre du projet, date, nombre d'events
- [ ] Nommage : `{projet}-storyboard-{date}.pdf`

## Implémentation suggérée

- Librairie `jspdf` + `jspdf-autotable` (légères, client-side)
- `src/lib/utils/export-pdf.ts` — fonction pure `generateStoryboardPDF(model)`
- Tests unitaires sur la génération de contenu (mock jspdf)
