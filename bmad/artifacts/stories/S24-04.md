# S24-04 — Intégration webhook ComfyUI / A1111

**Sprint:** 24
**Points:** 13
**Priorité:** 🔴 Basse (post-MVP)

## Contexte

Les utilisateurs avancés veulent déclencher la génération directement depuis latent-line sans copier-coller les prompts dans leur interface SD.

## Objectif

Intégrer une connexion optionnelle à une instance locale ComfyUI ou Automatic1111.

## Critères d'acceptance

- [ ] Configuration dans Settings : URL du serveur + API key (optionnel)
- [ ] Bouton "Generate" sur chaque event → envoie le prompt au serveur configuré
- [ ] Indicateur de progression (waiting / generating / done)
- [ ] Image générée affichée en miniature dans l'event card
- [ ] "Generate All" : envoi séquentiel de tous les events avec rate limiting
- [ ] Gestion des erreurs : timeout, serveur indisponible, API key invalide
- [ ] Stockage des images en `IndexedDB` (pas localStorage — trop grand)

## Implémentation suggérée

- `src/lib/services/ai-backend.ts` — abstraction ComfyUI / A1111
- `src/lib/stores/generation.svelte.ts` — état de génération par event
- Worker pour ne pas bloquer l'UI pendant la génération
