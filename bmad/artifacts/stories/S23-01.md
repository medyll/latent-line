# S23-01 — Partage de projet via URL

**Sprint:** 23
**Points:** 5
**Priorité:** 🟡 Moyenne

## Contexte

Il n'existe aucun moyen de partager un projet avec un collaborateur sans envoyer un fichier JSON manuellement. Un lien URL serait beaucoup plus simple.

## Objectif

Encoder le modèle JSON dans l'URL (hash) pour permettre le partage direct via lien.

## Critères d'acceptance

- [ ] Bouton "Partager" dans la topbar → génère une URL avec hash `#model=<base64-gzip>`
- [ ] L'URL est copiée dans le presse-papier avec un toast de confirmation
- [ ] Ouverture d'une URL avec hash → charge automatiquement le modèle partagé
- [ ] Si modèle local existant : modale de confirmation avant remplacement
- [ ] Limite : avertissement si le modèle > 8KB compressé (URL trop longue)
- [ ] QR code optionnel (librairie légère `qrcode-svg`)

## Implémentation suggérée

- `src/lib/utils/url-share.ts` — encode/decode base64 + gzip (`CompressionStream`)
- Lecture du hash dans `+page.svelte` via `onMount`
