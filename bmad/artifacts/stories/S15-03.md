# S15-03 — Nouveau projet / Reset

**Sprint:** 15
**Points:** 2
**Priorité:** 🟡 Moyenne

## Contexte

Il n'existe pas de moyen de repartir d'un projet vide sans vider manuellement localStorage. L'utilisateur reste bloqué sur le modèle exemple.

## Objectif

Ajouter une action "Nouveau projet" qui réinitialise le modèle et l'historique.

## Critères d'acceptance

- [ ] Bouton "Nouveau projet" dans la topbar ou menu Fichier
- [ ] Confirmation modale avant reset (données non sauvegardées perdues)
- [ ] Réinitialise `model` vers un modèle vide (pas l'exemple)
- [ ] Vide l'historique undo/redo
- [ ] Supprime la clé `latent-line:model` du localStorage
- [ ] Modèle vide : 0 assets, 0 events, config par défaut

## Implémentation suggérée

- Fonction `newProject()` dans `+page.svelte`
- `src/lib/model/model-empty.ts` — modèle vide validé par Zod
- Modale de confirmation réutilisable `ConfirmDialog.svelte`
