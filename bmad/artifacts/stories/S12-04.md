# S12-04 — Settings panel

**Sprint:** 12
**Points:** 3
**Priorité:** 🟠 Haute

## Contexte

Il n'existe aucune interface de configuration du projet. Le nom du projet, le FPS cible,
et d'autres paramètres sont figés dans le modèle sans UI dédiée.

## Objectif

Créer un panneau Settings accessible depuis la topbar pour configurer les métadonnées
du projet et les préférences de l'éditeur.

## Critères d'acceptance

- [ ] Icône ⚙️ dans la topbar ouvre un panneau settings (overlay ou panneau latéral)
- [ ] **Section "Projet"** :
  - Nom du projet (éditable, persiste dans `model.config.name`)
  - FPS cible (number input, persiste dans `model.config.fps`)
  - Description (textarea optionnelle)
- [ ] **Section "Éditeur"** :
  - Thème (light / dark) — remplace le toggle actuel de la topbar
  - Unité de temps affichée (frames / secondes / timecode)
- [ ] Les changements sont appliqués en temps réel et persistés via S12-01
- [ ] Fermeture par Échap ou clic hors du panneau
- [ ] Tests unitaires sur les mutations config

## Notes

- `model.config` existe déjà dans le schéma, vérifier les champs disponibles dans model-types.ts
- Si certains champs (`fps`, `name`) n'existent pas encore dans le type, les ajouter
