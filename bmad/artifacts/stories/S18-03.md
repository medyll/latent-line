# S18-03 — Export prompts AI (TXT / JSON)

**Sprint:** 18
**Points:** 3
**Priorité:** 🟠 Haute

## Contexte

Le but principal de latent-line est de préparer des séquences pour la génération vidéo AI. L'export des prompts formatés est le livrable central du workflow.

## Objectif

Exporter tous les prompts de la séquence dans un format directement utilisable par les outils AI (Deforum, ComfyUI, etc.).

## Critères d'acceptance

- [ ] Export "Prompts TXT" : un prompt par ligne, format `{frame}: {prompt complet}`
- [ ] Export "Prompts JSON" : array d'objets `{ frame, prompt, negative_prompt, seed, steps }`
- [ ] Prompt complet = composition de : action + personnage + mood + environnement + FX
- [ ] Option : inclure ou exclure les champs négatifs
- [ ] Option : format Deforum (`"0": "prompt", "24": "prompt"`)
- [ ] Preview du format avant export

## Implémentation suggérée

- `src/lib/utils/export-prompts.ts` — fonctions pures par format
- `ExportModal.svelte` — sélection du format + preview + download
