# S17-01 — Markers / repères nommés sur la timeline

**Sprint:** 17
**Points:** 3
**Priorité:** 🟡 Moyenne

## Contexte

Il n'existe aucun moyen de marquer des points clés dans la timeline (début d'acte, coupure, etc.). Les projets longs deviennent difficiles à naviguer.

## Objectif

Permettre d'ajouter des markers nommés et colorés sur la timeline temporelle.

## Critères d'acceptance

- [ ] Double-clic sur la règle temporelle crée un marker
- [ ] Marker affiche un label éditable et une ligne verticale colorée
- [ ] Clic droit sur un marker → menu contextuel : Renommer / Supprimer / Changer couleur
- [ ] Les markers sont persistés dans `model.timeline` (nouveau champ `markers: Marker[]`)
- [ ] Navigation par markers : Ctrl+Alt+→/← saute au marker suivant/précédent
- [ ] Markers inclus dans l'export JSON

## Implémentation suggérée

- Type `Marker: { id, time, label, color }` dans `model-types.ts`
- Rendu dans `SequenceOrchestrator.svelte` — couche superposée au track
