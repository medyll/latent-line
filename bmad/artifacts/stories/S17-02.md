# S17-02 — Multi-sélection d'events

**Sprint:** 17
**Points:** 5
**Priorité:** 🟠 Haute

## Contexte

Les opérations (supprimer, dupliquer, déplacer) ne s'appliquent qu'à un event à la fois. Pour les projets de 20+ events, c'est très lent.

## Objectif

Permettre la sélection multiple d'events et les opérations batch.

## Critères d'acceptance

- [ ] Shift+Clic sélectionne une plage d'events
- [ ] Ctrl+Clic toggle la sélection d'un event individuel
- [ ] Ctrl+A sélectionne tous les events
- [ ] Barre d'actions contextuelle apparaît quand ≥2 events sélectionnés : Supprimer / Dupliquer / Décaler
- [ ] "Décaler" déplace tous les events sélectionnés de N frames (input prompt)
- [ ] Visuel : outline bleu sur chaque event sélectionné
- [ ] Escape désélectionne tout

## Implémentation suggérée

- `selectedEventIds: Set<string>` dans le store de sélection
- `SelectionToolbar.svelte` — barre flottante au bas de l'écran
