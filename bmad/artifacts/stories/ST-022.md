# ST-022 — Scroll sync: Synoptic View ↔ Temporal Sequencer

- sprint: 6
- created: 20260308T
- status: ready
- points: 5

## Description

Implement scroll synchronization between Synoptic grid view and Temporal Sequencer. When user scrolls either pane, the other scrolls to match. Handles both horizontal (event timeline) and vertical (event rows) scroll.

## Acceptance Criteria

- Synoptic View and Temporal Sequencer share scroll state via context
- Horizontal scroll: both panes track `scrollLeft`
- Vertical scroll: both panes track `scrollTop`
- Scroll event debounced (100ms) to prevent performance degradation
- Unit tests for scroll sync logic (5 tests)
- E2E test: scroll Synoptic, verify Temporal follows (1 test)

## Implementation Notes

- Create `ScrollSyncStore` context key in `keys.ts` exporting `{scrollLeft, scrollTop}` as writable stores
- Both components `bind:scrollLeft` and `bind:scrollTop` to the stores
- Debounce via `throttle` utility in `src/lib/utils/` (if not exists, add)
- Temporal Sequencer container: `overflow: auto` with scroll listeners
- Synoptic grid: `overflow: auto` with scroll listeners
- Test: `SequenceOrchestrator.test.ts` +5 tests for sync behavior
