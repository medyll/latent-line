# S13-02 — Raccourcis clavier

**Sprint:** 13
**Points:** 3
**Priorité:** 🟡 Moyenne

## Contexte

L'application est entièrement pilotable à la souris. L'absence de raccourcis clavier
nuit fortement à la productivité pour un outil d'édition.

## Objectif

Implémenter les raccourcis clavier essentiels pour les actions les plus fréquentes.

## Critères d'acceptance

- [ ] `Delete` / `Backspace` — supprime l'event ou l'asset sélectionné
- [ ] `D` — duplique l'event sélectionné
- [ ] `Escape` — désélectionne tout / ferme les panneaux ouverts
- [ ] `Ctrl+Z` / `Ctrl+Y` — undo/redo (coordonné avec S13-01)
- [ ] `Ctrl+S` — force-save (déclenche l'export localStorage)
- [ ] `Ctrl+E` — déclenche l'export JSON (S12-02)
- [ ] `Space` — toggle play/pause du playhead
- [ ] `?` ou `F1` — affiche un overlay récapitulatif des raccourcis
- [ ] Les raccourcis sont désactivés quand le focus est dans un input/textarea
- [ ] Tests unitaires sur le handler de raccourcis

## Implémentation suggérée

- Listener global `keydown` dans `+page.svelte` ou `+layout.svelte`
- Vérification `event.target instanceof HTMLInputElement` avant d'agir
- Store `selectedEvent` et `selectedAsset` pour cibler les actions Delete/D
