# S16-03 — Miniatures d'events dans les cards

**Sprint:** 16
**Points:** 3
**Priorité:** 🟡 Moyenne

## Contexte

Les cards d'events dans la synoptic view affichent du texte brut. Une représentation visuelle (couleur par mood, avatar personnage) rendrait la lecture de la timeline plus rapide.

## Objectif

Enrichir visuellement les cards d'events avec couleurs mood, badge personnage et indicateurs d'état.

## Critères d'acceptance

- [ ] Couleur de fond de la card selon le mood (palette définie : joyful=vert, sad=bleu, etc.)
- [ ] Badge avec initiale(s) du personnage actif
- [ ] Icône si l'event a de l'audio, du dialogue, un effet FX
- [ ] Indicateur de durée (barre de progression proportionnelle)
- [ ] Mode "compact" (masque les détails texte) toggleable

## Implémentation suggérée

- Constante `MOOD_COLORS` dans `src/lib/model/mood-palette.ts`
- Modification de `TimelineEvent.svelte`
- Nouveau slot optionnel dans `SequenceOrchestrator`
