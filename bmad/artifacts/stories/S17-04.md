# S17-04 — Duplication et templates d'events

**Sprint:** 17
**Points:** 3
**Priorité:** 🟡 Moyenne

## Contexte

Créer un event similaire au précédent oblige à tout resaisir. La duplication accélérerait considérablement la production de longues scènes répétitives.

## Objectif

Dupliquer un event existant et créer des templates réutilisables.

## Critères d'acceptance

- [ ] Bouton "Dupliquer" sur chaque event card (Ctrl+D)
- [ ] L'event dupliqué est inséré juste après l'original avec time+1
- [ ] Menu "Sauvegarder comme template" dans le menu contextuel de l'event
- [ ] Panneau "Templates" dans l'AssetManager (nouvel onglet)
- [ ] Clic sur un template → crée un event avec ses valeurs
- [ ] Templates persistés dans `localStorage['latent-line:templates']`

## Implémentation suggérée

- `src/lib/stores/templates.svelte.ts`
- `src/lib/components/app/TemplatesPanel.svelte`
