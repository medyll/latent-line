## Important Issues

### 1. `console.log` calls in production component (AssetManager.svelte)

**Files:** `src/lib/components/workspace/assets/AssetManager.svelte` lines 117, 168, 211, 224, 256, 262
Debug logs left in production code. Should be removed before any release.

### 2. `SequenceOrchestrator.svelte` — mutable native `Set` instead of `SvelteSet`

**File:** `src/lib/components/workspace/orchestrator/SequenceOrchestrator.svelte:178`
Using a native `Set` breaks Svelte 5 reactivity. Selection state may silently not update the UI.
**Fix:** Replace `new Set(...)` with `new SvelteSet(...)` from `svelte/reactivity`.

### 3. Unused props in `TimelineEvent.svelte`

`zoom` and `timelineFrame` declared in `$props()` but never consumed (lines 22). Dead interface surface — either use them or remove from the prop signature.

### 4. `SnapshotsPanel.svelte` — unused import + `any` type

`deserializeModel` imported but never called (line 11). Indicates dead code from a partial refactor.
