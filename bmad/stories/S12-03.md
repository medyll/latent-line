## Objectif

Permettre à l'utilisateur d'importer un fichier JSON et de remplacer le modèle courant.

## Acceptance Criteria

- `handleImport()` lit le fichier, parse le JSON, valide via Zod
- Si valide : remplace project, assets, timeline, config dans le store
- Si invalide : affiche les erreurs dans `importErrors`, ne mute pas le modèle
- Round-trip complet : export → re-import → modèle identique
- Tests unitaires couvrent : import valide, import invalide, round-trip
