01 · Overview

WHAT IT IS

CapCut AI Studio est le pivot stratégique de ByteDance : transformer CapCut d'un éditeur vidéo timeline-based en une plateforme de création AI-native, centrée sur un canvas infini.

Le produit fusionne deux stacks distinctes : Video Studio (canvas-driven, lancé mars 2026) et AI Lab / Design Agent (génération image/vidéo inline). L'ensemble est propulsé par deux modèles propriétaires — Seedance 2.0 pour la vidéo, Seedream 5.0 pour l'image.

Positionnement : entre Runway (pro generative) et Adobe (full suite), avec un pricing agressif free-tier ciblant la creator economy et le commerce e-commerce.

03 · Fonctionnalités

FEATURE MAP

VIDEO STUDIO
⬚
Canvas

Infinite Canvas

Workspace sans timeline. Remplace Premiere Pro par un whiteboard Miro-like. Chaque card = un shot indépendant, promptable séparément. Non-linéaire par design.

◉
Consistency

Omni Reference

Ancrage visuel cross-shot : personnages, style guide, références de scènes précédentes injectés dans le pipeline de génération. Résout le drift entre clips.

⟿
Génération

AI Agent (Ideation)

Agent intégré qui brainstorme, rédige le script, structure en actes. Sortie : storyboard prêt à générer. Évite le passage par un LLM externe.

◈
3 Project Types

Canvas / Storyboard / Auto

Canvas Project : libre. Storyboard Project : séquencé avec arcs narratifs. Auto-Generated Project : input script → output vidéo complète en un click.

▣
Multi-modal

Seedance 2.0 Inputs

Jusqu'à 9 images + 3 vidéos + 3 audios en référence simultanée. Director-level controls : lighting, camera movement, character consistency.

◎
Post-gen

Toolkit Intégré

Object removal, background change, motion tracking, audio adjustment — sans quitter le canvas. Full editing loop dans une seule interface.

AI LAB / DESIGN AGENT
✦
Image Gen

Seedream 5.0 Design

Chat-style prompt interface. Text-to-image + image-to-image. Génère posters, ads, logos, covers en secondes. #1 Artificial Analysis Text-to-Image Leaderboard.

⟲
Conversational

Iterative Refinement

Affinage en langage naturel post-génération : "Keep the character but change the background to sunset". Pas de re-prompt complet nécessaire.

◐
Commerce

Commerce Pro

URL-to-Video (Shopify link → promo vidéo). AI Model Try-On (vêtements sur avatars). Batch product visuals. ~26–30$/mois, cible e-commerce / dropshippers.

⧉
AI Core Tools

Auto captions · AutoCut · TTS

Features les plus utilisées en 2025. AutoCut assemble les clips en clean edits. Voice filter + TTS pour narration sans setup complexe. Lip sync natif.

◫
Try-On

AI Try-On & Avatar

100+ avatars digitaux. Clone d'avatar personnalisé via upload vidéo courte. Génère du voiceover sans apparaître à caméra. Fashion product without photoshoot.

⟁
Safety

C2PA + Guardrails

Watermarking cryptographique C2PA embedded dans chaque vidéo générée. Blocage real faces / IP protégée. Conformité pro-active aux régulations à venir.

DESIGN LANGUAGE — Principes identifiables

Spatial

Canvas infini = espace sans contrainte. Rupture avec le paradigme temporel linéaire. Spatialité = liberté créative non-linéaire.

Densité contrôlée

UI dark, haute densité informationnelle dans les panels, espace généreux dans le canvas central. Hiérarchie claire pro → amateur.

Card-as-unit

Chaque shot = une card indépendante promptable. Unité atomique de création. Pattern emprunté au design system Figma / Notion.

Gradient AI glow

Éléments AI-générés marqués par un halo coloré (purple/teal). Signal visuel clair : humain vs AI-generated. Pattern émergent industrie.

07 · Roadmap & Historique

CapCut v1 — Timeline Editor
Éditeur mobile ByteDance. Succès viral sur TikTok. Interface timeline classique. Templates, effets, musique intégrée.

Web Platform + Première IA

Tier B2B ~26-30$/mois. URL-to-Video, AI Try-On, commercial licensing. Seedream 5.0 classé #1 benchmark T2I. AI Lab stabilisé. Billions de projets créés sur l'année.

Lancement Video Studio (web). Seedance 2.0 intégré. Infinite canvas, omni reference, AI agent idéation. 3 project types. Disponible SEA/MENA/LATAM/Afrique. 150 crédits/semaine gratuits.

Extension EU/US (IP concerns à résoudre). Clips >15s. Collaboration temps-réel sur canvas. Inputs multi-modal élargis. API Volcano Engine publique déjà disponible pour intégrations tiers.

09 · Risques & Limites

RISQUES & WATCH POINTS

🔴 Élevé

Restriction géographique Seedance 2.0

Pas EU/US pour l'instant (IP enforcement). Bloque l'accès à la feature principale pour les marchés principaux. Délai d'expansion inconnu.

🔴 Élevé

Contexte réglementaire ByteDance US

JV Oracle en cours. Loi Foreign Adversary App toujours active. Risque de nouvelles restrictions sur le marché US influençant l'adoption globale.

🟡 Modéré

Limite 15s / clip

Content long-form impossible sans assemblage manuel de clips courts. Frein pour usage film/narrative sérieux. Runway et Sora vont au-delà.

🟡 Modéré

Propriété intellectuelle générée

C2PA watermark embarqué dans tous les outputs. Statut légal du contenu AI-généré en évolution rapide dans toutes les juridictions.

🟡 Modéré

Dépendance Volcano Engine

Toute l'IA repose sur l'infra ByteDance. Pas d'alternative si restriction ou changement de policy. API publique mitige partiellement.

🟢 Faible

Free tier credits conversion

150 crédits/semaine suffisants pour tester mais limite la production à scale. Conversion free→Pro non encore prouvée à grande échelle.
